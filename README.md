# 🎧 MoodTune

Application web qui génère une playlist musicale personnalisée selon ton humeur.

## 🚀 Lancer l'application

**Prérequis :** Node.js installé ([nodejs.org](https://nodejs.org))

```bash
# 1. Cloner le dépôt
git clone https://github.com/amine95-cloud/moodtune.git

# 2. Créer un projet React
npx create-react-app mon-moodtune
cd mon-moodtune

# 3. Remplacer src/App.js par le contenu de moodtune.jsx

# 4. Lancer
npm start
```

Ouvre **http://localhost:3000** dans Chrome ou Firefox.
